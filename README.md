# digital-circuits



\# Digital Circuits (DI)



This repository contains my laboratory work for the course \*\*Digital Circuits (DI)\*\* at HAW Hamburg.



The course focuses on practical digital design using VHDL, FPGA implementation, timing analysis, and programmable architectures.



---



\## 🔧 Technologies \& Tools



\- VHDL

\- Vivado Design Suite

\- FPGA: Artix-7 (XC7A100T / XC7A35T)

\- MODSYS 2.0 Evaluation Board

\- Basys 3 Board

\- Behavioral Simulation \& Timing Analysis
